iOS 9 Search Feature

###iOS 9 Search
在WWDC2015大会上苹果隆重介绍了iOS9的Search功能，以前只能搜搜邮件，Safari History以及短信的Search功能现在完全整合了Spotlight之后，可以搜索任意App配置的内容，如果你的App有自己的Web网站的话，还可以通过Safari来搜索web网站上配置的内容。可以说是非常强大。

Apple在iOS9上推出了三种搜索方式:

  • NSUserActivity
  • CoreSpotlight
  • Web Markup
NSUserActivity
  • 支持Spotlight and Safari(if NSUserActivity support web url) Search
  • Benifit HandOff and Siri
  • Default Private Index, Allow set item to Public Index

NSUserActivity只能用于HandOff功能，也就是说你本机配置了之后，本机是搜索不到该配置的内容的，只能通过HandOff将搜索结果传递到其他iOS设备上，通过Spotlight和Safari进行搜索。

NSUserActivity还有个重要的功能就是容许将搜索结果公开到苹果服务器，服务器通过算法得出搜索结果是否有效，这样其他用户在搜索的时候可以自动匹配到最佳的结果。

####CoreSpotlight

  • Add, Update, Delete item

aTranslator


aTranslator是一款基于百度翻译的翻译小工具，具备翻译以及词典的功能，支持10多种语言相互间翻译，特别支持文言文以及粤语翻译，让你一秒钟写诗不再是梦。

写一个翻译App的初衷一直都有，作为程序猿工作中难免会遇到一些不认识的单词，以前google没有被墙的时候用google translate就感觉特别麻烦，所以就想是否可以写一款基于google的app放在桌面上呢？等到自己动手开始做得时候google已经被gfw了，所以就只能先写了这款基于百度翻译的。百度翻译也有它强大的优势，比如支持关联百科，丰富的双语以及柯林斯例句，还有网络释义等等，在适应中国人学习英语的习惯这方面还是做得非常好的，目前aTranslator其实已经有解析这些东西了，但是还没有将功能开发出来，在后面的版本迭代里面会陆续提供上面说得那些功能。

后来小伙伴提醒到，谷歌翻译中国 还可以继续使用，研究了一下，后面可能会考虑将google翻译的结果纳入到aTranslator里面。

目前aTranslator还在App Store审核之中，等到上线到Store的时候，我会更新一下下载链接。如果你有任何问题，可以在下面留言，我第一时间会回复你。

aTranslator已经登陆Mac App Store，如果你有兴趣，可以下载一个试试。
