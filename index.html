Hider

Hide your Applications, Keep your Privacy and Make you Focus

Features

Batch hiding of applications in groups

Support creating different application groups to hide applications, including message notifications and resource files associated with the applications, meeting the hiding needs in different scenarios to protect your privacy.

Scheduled execution of hiding applications

Support creating scheduled tasks to hide applications. When the task ends, the applications will be displayed again, helping you stay focused and undisturbed!

Screen Widget

Support screen widgets for quickly hiding and displaying applications.

Data statistics

Track current and historical hiding records, as well as the number of scheduled task occurrences and total time.

Password protection

When you enter the Hider app, you need to enter a password or verify with Face ID to be allowed in. Additionally, it supports a decoy password to prevent the need to enter the real password in special circumstances, protecting your privacy.

Prohibit installation and uninstallation

Prohibit installation of applications from the App Store or deletion of the applications that you have already installed.
