iOS 9 Search Feature

iOS 9 Search

在WWDC2015大会上苹果隆重介绍了iOS9的Search功能，以前只能搜搜邮件，Safari History以及短信的Search功能现在完全整合了Spotlight之后，可以搜索任意App配置的内容，如果你的App有自己的Web网站的话，还可以通过Safari来搜索web网站上配置的内容。可以说是非常强大。

Apple在iOS9上推出了三种搜索方式:

  • NSUserActivity
  • CoreSpotlight
  • Web Markup
NSUserActivity
  • 支持Spotlight and Safari(if NSUserActivity support web url) Search
  • Benifit HandOff and Siri
  • Default Private Index, Allow set item to Public Index

NSUserActivity只能用于HandOff功能，也就是说你本机配置了之后，本机是搜索不到该配置的内容的，只能通过HandOff将搜索结果传递到其他iOS设备上，通过Spotlight和Safari进行搜索。

NSUserActivity还有个重要的功能就是容许将搜索结果公开到苹果服务器，服务器通过算法得出搜索结果是否有效，这样其他用户在搜索的时候可以自动匹配到最佳的结果。
